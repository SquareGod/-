# -
基于RAG的通用文档分析智能问答系统，旨在帮助用户快速从文档中获取准确信息。
测试文档为"电子鼻的算法设计综述简要回答.pdf"
构建通用文档的智能问答分析系统，替代人工查阅文档的低效模式，信息获取效率提升显著。
自定义加载器，支持PDF/Word/CSV 等多种格式文档解析，结合文本递归分块策略，解析准确率达 80%+；
构建 Chroma+BM25 混合检索体系，叠加多种优化策略，检索精度、召回率较单一检索分别提升 35%、28%；
基于 SQL 数据库实现知识库索引去重与增量更新，构建效率提升显著，集成多轮对话记忆与流式输出；
使用 RAGAS 搭建量化评估体系，检索精度、答案忠实度与相关性等核心指标较基础方案提升 30%-42%；
基于 Gradio 框架落地 Web UI，支持模型参数（温度 / 最大长度）调节、多知识库切换、流式输出。
